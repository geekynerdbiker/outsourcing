다음 사이트에서 linear regression과 deep learning 기반의 regression에 적합한 의미 있는 데이터 셋 하나를 선택해서 분석을 하시오 (데이터 크기는 너무 작지 않아야 하고 충분한 수의 입력 특성이 있어야 함)
- 국가통계포털 (https://kosis.kr/index/index.do)

1. 보고서에 특정한 데이터 셋을 선택한 이유 기술 (어디서 내려 받은 어떤 데이터인지 상세하게 설명)
2. 데이터를 linear regression model로 분석 (learning rate와 같은 값을 변경해서 loss가 변화하는 것을 보고서에 기재)
3. 데이터를 deep learning 기반의 regression model로 분석 (learning rate와 같은 값을 변경해서 loss가 변화하는 것을 보고서에 기재)
4. linear regression과 deep learning 기반의 regression의 loss 값 비교하고 분석한 것을 보고서에 기재
5. 보고서, 데이터 파일, linear regression과 deep learning 기반의 regression의 ipynb 파일을 하나의 zip file로 만들어 제출 (교수자가 zip 파일을 하나의 directory에 풀었을 때 문제 없이 실행할 수 있어야 함. 실행이 되지 않으면 0점 처리)

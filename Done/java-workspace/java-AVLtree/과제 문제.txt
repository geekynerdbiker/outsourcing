오픈소스인 C++ STL(Standard Template library)의 set의 관련 기능들을 구현한다. set은 자료구
조 수업에서 배운 AVL 트리를 이용하여 구현해야 한다. 제시된 set의 기능(함수)들은 과제의
요구사항을 준수해야 하고, 효율성을 고려하여 구현해야 한다.

1. 구현할 기능 (사용자 요구사항)
1) empty: set이 비어 있으면 1을 출력하고, 비어 있지 않으면 0을 출력한다.
2) size: set에 저장된 원소의 수를 출력한다.
3) find x: 노드 x의 depth를 출력한다. 만약 노드 x가 없다면, 0을 출력한다.
4) insert x: 새로운 노드 x를 삽입하고, 노드 x의 depth를 출력한다. 이때 x는 현재 set에
존재하지 않은 값으로 주어진다.

2. 입출력 형식
1) 입력
첫 번째 줄에는 테스트 케이스 수 𝑇 (1 ≤ 𝑇 ≤ 5)가 주어진다. 이후 각 테스트 케이스의 정
보가 다음과 같이 주어진다.
• 첫 번째 줄에 명령어의 수 𝑄 (1 ≤ 𝑄 ≤ 100,000)가 주어진다. 이후 𝑄개의 줄을 통해
기능들을 수행할 입력들이 차례로 주어진다.
2) 출력
각 명령어의 요구사항대로 출력 내용을 한 줄에 출력한다. 단, 출력할 내용이 2개 이상이면
공백으로 구분한다.
